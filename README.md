# iOSゼミへようこそ！
## 目的
このプロジェクトは、Swift言語に触れたことがない方向けへの参考例のプロジェクトです。  
プロジェクトはPlaygroundsを利用することで簡単にすばやくコードを実行することを実現しています。  
**今すぐ、コードをクローンしXcodeで実行してください！**

## 注意
以下のバージョンでのみ動作確認しております。
- Xcode : 12.3 
- Swift : 5.3.2
